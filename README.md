# 中考体育分数预测模型 (2.0版本)

## 🎯 项目概述

基于真实的中考体育分数数据，构建了一个高精度的机器学习模型，用于预测学生的中考体育分数。模型使用5000条真实学生体育测试数据训练，预测精度极高。

## 📊 模型性能

### 核心指标
- **测试集 R²分数**: 0.9711
- **测试集 MAE**: 0.7166分
- **测试集 RMSE**: 1.4012分
- **交叉验证 R²**: 0.9641 (±0.0059)

### 准确率分析
- 误差 ≤ 1分的准确率: 81.4%
- 误差 ≤ 2分的准确率: 91.0%
- 误差 ≤ 3分的准确率: 95.7%
- 误差 ≤ 5分的准确率: 98.8%
- 等级预测准确率: 96.5%

## 🏆 特征重要性

根据随机森林模型分析，最重要的特征包括：

1. **标准分数** - 47.8%
2. **50米跑分数** - 11.2%
3. **跳绳个数** - 8.5%
4. **跳绳分数** - 8.1%
5. **坐位体前屈成绩** - 6.6%

### 3. 直接使用模型
```python
import joblib
import numpy as np

# 加载模型
model = joblib.load('中考体育分数预测模型.model.pkl')
scaler = joblib.load('中考体育分数预测模型.scaler.pkl')
feature_names = np.load('中考体育分数预测模型.features.npy', allow_pickle=True)

# 准备输入数据
input_data = {
    '身高_cm': 150.0,
    '体重_kg': 45.0,
    '肺活量_ml': 2000.0,
    '50米跑_秒': 10.0,
    '坐位体前屈_cm': 12.0,
    '跳绳_个': 80.0,
    'BMI分数': 75.0,
    '体重分数': 85.0,
    '肺活量分数': 75.0,
    '50米跑分数': 75.0,
    '坐位体前屈分数': 75.0,
    '跳绳分数': 75.0,
    '标准分数': 75.0
}

# 确保特征顺序正确
X = np.array([[input_data[feature] for feature in feature_names]])

# 标准化
X_scaled = scaler.transform(X)

# 预测
score = model.predict(X_scaled)[0]
print(f"预测中考体育分数: {score:.1f}")
```

## 📝 输入数据说明

### 需要输入的13个特征：

1. **身高_cm** - 身高 (厘米)
2. **体重_kg** - 体重 (公斤)
3. **肺活量_ml** - 肺活量 (毫升)
4. **50米跑_秒** - 50米跑成绩 (秒)
5. **坐位体前屈_cm** - 坐位体前屈成绩 (厘米)
6. **跳绳_个** - 跳绳个数 (个)
7. **BMI分数** - BMI评分 (0-100分)
8. **体重分数** - 体重评分 (0-100分)
9. **肺活量分数** - 肺活量评分 (0-100分)
10. **50米跑分数** - 50米跑评分 (0-100分)
11. **坐位体前屈分数** - 坐位体前屈评分 (0-100分)
12. **跳绳分数** - 跳绳评分 (0-100分)
13. **标准分数** - 标准分 (0-100分)

### 建议输入范围：
- 身高: 小学生 120-160cm, 初中生 140-180cm
- 体重: 小学生 20-50kg, 初中生 30-70kg
- 肺活量: 小学生 800-2500ml, 初中生 1500-4000ml
- 50米跑: 小学生 9-13秒, 初中生 7-11秒
- 坐位体前屈: 0-25cm
- 跳绳: 30-150个
- 各项分数: 0-100分

## 🔧 技术实现

### 数据处理流程
1. **数据清洗**: 处理缺失值，使用中位数填充
2. **特征选择**: 选择13个最相关的特征
3. **标准化**: 使用StandardScaler标准化特征
4. **模型训练**: 使用随机森林回归算法
5. **性能验证**: 交叉验证和全数据集验证

### 模型参数
- 算法: 随机森林回归
- 树数量: 200棵
- 最大深度: 20层
- 特征选择: sqrt(特征数)
- 随机种子: 42
